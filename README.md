🎮 Tic Tac Toe AI Bot

An interactive Tic Tac Toe game with an intelligent AI opponent, built using HTML, CSS, and JavaScript. This project focuses on combining game logic with a smooth and engaging user experience.

🚀 Features
🎯 Three Difficulty Levels
Easy
Moderate
Hard (Smart AI gameplay)
🤖 AI Opponent
Makes decisions based on selected difficulty level
🔊 Sound Effects
Win sound
Lose sound
Draw sound
🏆 Dynamic Result Display
“You Win”
“AI Wins”
“Game Draw”
🔄 Restart Game
Reset the board instantly and play again
🧠 Concepts Used
DOM Manipulation – Updating UI dynamically
Game Logic Implementation – Win patterns & conditions
JavaScript Decision Making – AI moves
Event Handling – User interactions (clicks)
Responsive UI Design
🛠️ Tech Stack
HTML
CSS
JavaScript
🎯 Project Objective

The goal of this project is to build a fun and interactive game while strengthening core concepts of JavaScript logic, DOM manipulation, and UI handling.
